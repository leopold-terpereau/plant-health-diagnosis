Démarche à suivre pour utiliser l'application :

Première utilisation :

    Dans le dossier local-server, supprimer le dossier node_modules et le fichier package-lock.json s'ils existent

    Dans un terminal, taper :
    cd TensorFlowJS\local-server
    npm install
    node server.js

    Dans votre navigateur préféré, taper l'URL :
    http://localhost:3/main.html

Vous pouvez utiliser l'application :)


Pour accéder de nouveau à l'application, il vous suffit de taper dans un terminal :

    cd TensorFlowJS\local-server
    node server.js

    Dans votre navigateur préféré, taper l'URL :
    http://localhost:3/main.html

Vous pouvez utiliser l'application :)


Note : une fois que vous avez fini, il ne vous reste plus qu'à tuer (icône Poubelle dans VSCode) ou quitter le terminal
